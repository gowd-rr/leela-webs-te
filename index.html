This is an H1 Heading

The <h1> tag is used for the most important heading, typically the main title of the webpage. It is the largest in size.

This is an H2 Heading

The <h2> tag is used for subheadings, typically for sections under the main title. It is slightly smaller than the H1 heading.

This is an H3 Heading

The <h3> tag is used for further subdivisions within an H2 section, smaller in size.

This is an H4 Heading

The <h4> tag is used for subsections within an H3 section. It is smaller in size, often used for smaller headings or subtopics.

This is an H5 Heading

The <h5> tag is used for additional subdivisions. It is rarely used unless there are many nested sections.

This is an H6 Heading

The <h6> tag is the smallest heading tag. It is used for minor subsections or notes that are less prominent.
